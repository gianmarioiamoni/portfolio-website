YelpCamp

Review campgrounds

YelpCamp is a web application that allows users to review and rate campgrounds and resorts worldwide. It's designed to help outdoor enthusiasts find the best camping spots based on user reviews and ratings.

Functionalities:

  • User registration and login with username/password or social accounts
  • View and rate campgrounds
  • Edit own campgrounds
  • Main dashboard with global map and campground details
  • Advanced search capabilities by keyword and location
  • View specific campground details and user reviews
  • Create and edit campground details

Authentication and Authorization

Users can register and log in using either a username/password or their social media accounts. Registered users can view and rate all stored campgrounds but can only edit those they have created.

Main Dashboard

The main dashboard provides a global map indicating all reviewed campgrounds. Users can access specific campground details and perform advanced searches.

Campgrounds Search

Users can search for campgrounds worldwide based on keywords or proximity to their current location.

Campground Details

Each campground's detail page provides information about the location and a list of reviews from all users.

Campground Edit and Creation

Users can edit the details of the campgrounds they have added. They can also create new campground entries, providing essential information and reviews. Newly created campgrounds will be available for all users to view.
