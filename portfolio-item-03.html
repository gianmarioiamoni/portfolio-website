Todos 2.1

Todos 2.1 is the new version of the Todos project. It is a web application based on the MERN stack that allows the creation of customized todo lists.

The backbone of the project is based on an idea of Colt Steele, from "The Web Developer Bootcamp 2024" course, available on Udemy. The main new features and improvements added are:

  • Migration to MongoDB database
  • New theme
  • New Homepage with responsive Appbar
  • Authentication with local username/password and Google ID
  • New navigation bar
  • Authorization
  • ALL TODOs list available for each user
  • Added Priority and Expiry date to todos and editing functionalities
  • Todos sorting and filtering capabilities
  • List deleting

New Authentication Framework

The new release includes a revamped authentication interface and logic, allowing users to sign up, log in, and log out, providing access to protected routes only for authenticated users.

User Profile Management

The application now includes a comprehensive user profile management feature, allowing users to upload and update profile pictures, change usernames, emails, and passwords. It also includes links for signing out and deleting the user profile.

New Navigation Bar

The application now features an improved navigation bar, available on every page, providing context-based commands to enhance user experience and accessibility.

ALL TODOs List, Priority, Expiry Date, and Creation Functionalities

Each user has an "ALL TODOs" list at the top of their available item lists, allowing them to view all tasks. Users can create their own lists, assigning names and icons to them.

Sorting and Filtering

Each list can be sorted either by priority and expiry date (default) or by expiry date only. Additionally, users can filter todo items to show only those due on the current day.
