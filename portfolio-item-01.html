Agile Project Manager

streamline and enhance
the management of agile projects

Agile Project Manager is a comprehensive web application designed to streamline and enhance the management of agile projects. It provides a suite of tools to facilitate project planning, task management, and team collaboration, ensuring projects are delivered on time and within scope.

Project Details

Authentication and authorization

User Registration and Login:

  • secure registration request via email and login processes for users
  • password hashing and salting
  • JWT (JSON Web Token) authentication and authorization

Project management

Creation, Modification, and Deletion of Projects

  • admins and authorized users can create new projects
  • admins can modify and delete existing projects
  • projects can be updated with new information or requirements
  • projetcs can be deleted if they are no longer needed

Assigmnent of team members to projects

  • admins and authorized users can assign team members to specific projects based on their role and responsability
  • team members receive a notification when they are assigned to a project

Project backlog

Creation and Management of Project Backlog

  • admins and authorized users can create new project backlog items
  • admins and authorized users can modify and delete existing project backlog items
  • project backlog items can be updated with new information or requirements
  • project backlog items can be deleted if they are no longer needed

Prioritization of Project Backlog items

  • authorized users can prioritize project backlog items

Drag and Drop functionality

  • intuitive drag and drop functionality to reorder project backlog items
  • users can drag and drop project backlog items to reflect changes in priority

Sprint Planning

Creation and Management of Sprints

  • users can create sprints, defining the duration and goals for each sprint
  • sprints help in organizing work into manageable chunks

Assignment of tasks to sprints

  • tasks from project backlog can be assigned to specific sprints
  • this assures that work is planned and executed in a structured manner

Monitoring tasks status and progress

  • users can track the status of tasks within each sprint
  • tasks can be marked as To Do, In Progress, or Done

Assignment of team members to tasks

  • resources can be assigned to specific tasks

Project Dashboard and Reports

Project details and status

  • the dashboard provides an overview of project information and status of sprints and tasks

User and roles management

User Account Creation and Management

  • admins can create and manage user accounts
  • new created user is notified by email about the login details
  • users can update their profiles and change passwords

Role Management

  • admins can modify user privileges to grant or restrict access to certain features
  • privileges can be customized based on the user's role and responsibilities
  • this allows for greater control over user access
